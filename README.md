
# Arduino audio visualizer

  ## Libraries used:
 - [ArduinoFFT](https://github.com/kosme/arduinoFFT)
 - [U8G2lib](https://github.com/olikraus/u8g2)
 
  ## Used components:
  - Arduino nano
  - 2x MAX7219
  - AUX connector
  - Microphone (Optional)
